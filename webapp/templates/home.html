Important information

PolyOligo is available open-source with a command line interface HERE. The standalone version can be used with any reference genome and VCF file, has more features and is faster because multi-core enabled.

To avoid overloading the server and transferring large amount of data, most methods come with limitations here. Notably, only a limited number of reference genomes are made available (current list: - more can be added upon request), the number of concurrent designs is capped and mutations from a VCF file cannot be considered (excl. strawberries).
