Important information

Note that PolyOligo is available as an open-source standalone command line software HERE. To avoid overloading the server, the scope had to be limited. Notably, the number of designs is capped and mutations (i.e. harnessing a VCF file) cannot be considered here (excl. strawberries). The standalone version is free, unlimited and much faster because multi-core enabled.

Also note that we do not use cookies and under no circumstances are we storing or using data that is uploaded to this server. Once a job completes, only results files are kept and this for 24h to ensure recovery before being permanently deleted.
